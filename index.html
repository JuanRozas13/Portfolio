MINHAS HABILIDADES

CONHECIMENTO BÁSICO

HTML BÁSICO

HTML (HyperText Markup Language) é a linguagem padrão para criação de páginas web. Utiliza tags como , e para estruturar o conteúdo, incluindo texto, imagens e links. Combinado com CSS (Cascading Style Sheets) para estilização e JavaScript para interatividade, forma a base para o desenvolvimento de páginas dinâmicas e interativas na internet.

CSS BÁSICO

CSS (Cascading Style Sheets) é uma linguagem de estilo que define a apresentação e o layout de elementos HTML em uma página web. Permite ajustar cores, fontes, margens, alinhamentos e outros aspectos visuais para criar designs atraentes e responsivos. Separado do HTML, o CSS simplifica a manutenção e a consistência visual em todo o site.

JAVA SCRIPT BÁSICO

JavaScript é uma linguagem de programação para web, adicionando interatividade, validação de formulários, animações e comunicação com servidores, tornando páginas mais dinâmicas e responsivas. Permite manipular elementos HTML, responder a eventos do usuário e criar aplicativos dinâmicos, como jogos e aplicativos de comércio.

GITHUB BÁSICO

GitHub é uma plataforma de hospedagem de código-fonte e colaboração, baseada em Git. Permite que desenvolvedores trabalhem em projetos, versionem código, acompanhem alterações, colaborem com outros e hospedem seus repositórios de forma pública ou privada.

GIT BÁSICO

Git é um sistema de controle de versão distribuído, usado para rastrear alterações em arquivos de código. Permite que desenvolvedores trabalhem em projetos colaborativos, gerenciem versões de código, revertam alterações e facilitem a colaboração em equipe.
